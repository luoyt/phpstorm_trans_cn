# 创建TODO项


**要创建TODO项**

1. 在编辑器打开目标文件，然后将插入符指向要创建TODO项的位置
2. [创建一个注释](/)，例如，你可以用`Ctrl+/`快捷键
3. 在注释中，输入匹配某个TODO模式的字符串。默认的，在任意位置以TODO开头的字符串(不管哪种情况)都被理解为TODO项并被高亮显示
4. 在TODO工具窗查看TODO项列表

# 使用导航栏跳转


使用[导航栏](/如何使用/常规指南/用户界面引导/导航栏.md)作为一种方便的工具来作为你在项目中查找的方式。

**要使用导航栏跳转到文件**

1. 按`Alt+Home`激活导航栏
2. 使用方向键或鼠标指针定位目标文件
3. 双击所选的文件，或按`Enter`来在编辑器中打开该文件

## <span id='基础'>基础</span>

填充你的项目通过创建新的各种元素(目录和文件)。PhpStorm建议下列可供选择的方法来访问相关功能：

* 主菜单的**File|New**
* 上下文菜单中的**New**命令
* 快捷键`Alt+Insert`

作为结果，将展示**New**菜单，在这你可以选择要被创建的元素类型。
